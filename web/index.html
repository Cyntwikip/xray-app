Data Source

The dataset of X-ray images is from the National Institutes of Health (NIH). There are 14 labels or classes, namely: Atelectasis, Cardiomegaly, Effusion, Infiltration, Mass, Nodule, Pneumonia, Pneumothorax, Consolidation, Edema, Emphysema, Fibrosis, Pleural Thickening, and Hernia.

Model Used

For the model, we adapted CheXNet, which is a 121-layer convolutional neural network. The model accepts an X-ray image and returns the probability of the detected thoracic disease. Moreover, we created a class activation map which outputs a heatmap of the region of interest from the image.

Other Applications

The model can also be used for any other machine vision projects especially those with regards to health.
